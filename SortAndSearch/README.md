# Sorting algorithms
## Usage: python sort.py [algorithm name]
Press [space] to start algorithm  
Press KEY [c] to clear
  
### Algorithms:
- bubblesort
- cocktailsort
- selectionsort
- insertionsort
- quicksort
- mergesort
